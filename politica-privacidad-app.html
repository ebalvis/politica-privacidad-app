Política de Privacidad

Esta aplicación recopila y utiliza datos de localización (mediante señales WiFi y sensores del dispositivo) con el único propósito de estimar la posición del usuario dentro de un edificio.

Datos recopilados

  • Intensidad de señal WiFi (RSSI)
  • Dirección MAC anonimizada de puntos de acceso
  • Coordenadas aproximadas de ubicación

Finalidad

Los datos se usan exclusivamente para:

  • Determinar la posición relativa del usuario en tiempo real
  • Mejorar la precisión mediante técnicas de interpolación y filtro de Kalman

No se recopilan

  • Datos personales identificables
  • Información de otros sensores no relacionados

Compartición de datos

Los datos no se comparten con terceros y toda la información se procesa localmente en el dispositivo.

Permisos utilizados

  • ACCESS_FINE_LOCATION: para acceder a la intensidad de señal WiFi (requisito del sistema Android).
  • ACCESS_WIFI_STATE: para leer la información de redes cercanas.

Seguridad

No se transmite ningún dato a servidores externos.
